Tag Selector

Used by selecting an element's tag (excluding the brackets)

Example Syntax:

HTML

p class="selector"

CSS

.selector

Class Selector

Assign an element a class. Also known as a "dot selector" due to it's syntax

Example Syntax:

HTML

p class="selector"

.selector

ID Selector

Assign an element an id. These selector's can only be used once so best to use sparingly.

Example Syntax:

HTML

p id="selector"

CSS

#selector
